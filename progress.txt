FRIDGE TO FORK — PROJECT PROGRESS

STRUCTURE
---------
Backend:    D:/Sofretna/fridge-to-fork           (Laravel 12 API, port 8000)
Frontend:   D:/Sofretna/fridge-to-fork-frontend  (React + Vite, port 5173)
Admin Panel: D:/Sofretna/fridge-to-fork-admin    (React + Vite, port 5174)

HOW TO RUN
----------
Terminal 1 (backend):
  cd D:/Sofretna/fridge-to-fork
  php artisan serve

Terminal 2 (frontend):
  cd D:/Sofretna/fridge-to-fork-frontend
  npm run dev

Terminal 3 (admin panel):
  cd D:/Sofretna/fridge-to-fork-admin
  npm run dev

Frontend:    http://localhost:5173
Admin Panel: http://localhost:5174

To make a user an admin:
  php artisan admin:make user@email.com
